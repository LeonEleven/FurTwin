### 新增功能

**动作库搜索：**
- 动作库新增名称搜索输入框，支持在当前类型 tab 内按名称快速筛选
- 搜索兼容批量选择模式，「全选」只作用于当前可见结果

- 右键菜单动作列表跟随排序后顺序

- 控制面板标题区域显示当前版本号
- Tray 右键菜单底部显示版本号

### FurTwin v0.1.0

FurTwin 是一款 Windows 桌面宠物软件，支持从绿幕视频提取透明动作帧，并管理桌面宠物动作库。

### 下载

| 产物 | 说明 | 适用人群 |
|------|------|----------|
| `furtwin-0.1.0-setup.exe` | 安装版 | 推荐普通用户下载，支持安装向导，可选择安装目录 |
| `furtwin-0.1.0-win-unpacked.zip` | 免安装版 | 推荐不想安装的用户下载，解压后运行 `furtwin.exe` |

### 主要功能

- 桌面宠物透明叠加，支持拖动和右键菜单
- 自定义动作库，支持导入、导出、重命名、删除
- 批量导入动作包（zip 格式）
- 自动行为，按间隔随机切换动作
- 点击触发，点击宠物播放互动动作
- 隐身模式 / 鼠标穿透，鼠标移到宠物区域时自动隐藏
- 提示词生成器，支持自定义动作描述
- 绿幕视频提取透明序列帧
- 系统托盘入口，支持快捷操作
- 开机自启

### 本次主要变化

- 安装包改为安装向导，支持选择安装目录
- 新增免安装 zip 产物
- 修复打包后 Tray 图标缺失
- 系统托盘入口、隐身模式、开机自启、自动行为等已完成回归测试
- 新增/更新文档：README、USER_GUIDE、RELEASE_CHECKLIST、PROJECT_AUDIT

### 使用注意事项

- 当前仅验证 Windows 系统
- 动作首次运行时需要计算 shape-cache，可能轻微卡顿，后续运行会流畅
- 开机自启可能受 Windows 权限或安全软件影响，如不生效请以管理员身份运行一次

### 问题反馈

反馈时请提供：
1. 操作步骤（如何复现问题）
2. 截图或录屏
3. 使用环境：安装版 / 免安装版 / 开发版
