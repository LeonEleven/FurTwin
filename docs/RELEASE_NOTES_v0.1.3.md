# FurTwin v0.1.3 Release Notes

---

## FurTwin v0.1.3 - 稳定性增强与体验优化

### Release 标题

```
FurTwin v0.1.3 - 稳定性增强与体验优化
```

### 版本信息

- **版本号**：v0.1.3
- **类型**：稳定性增强版本
- **基于 Commit**：`5fdb3b7`（fix: 清理退出阶段自动行为定时器）

### 新增功能

**配置保存可靠性增强：**
- 程序配置（动作排序、自动行为开关、自动行为参数等）采用原子写入方式保存，写入过程中异常退出不会损坏配置文件，降低用户配置丢失风险
- 启动时若检测到配置文件异常，会自动尝试从备份中恢复，避免影响用户已有的排序和设置

**找回桌宠入口：**
- 系统托盘右键菜单和桌宠右键菜单均新增「找回桌宠」选项
- 适用场景：桌宠被其他窗口遮挡、处于隐身模式、最小化或隐藏状态下，可通过该选项恢复桌宠可见并重新置顶
- 该操作不会改变桌宠当前位置

**主进程诊断日志：**
- 新增主进程运行日志，默认保存在：`%APPDATA%\FurTwin\logs\furtwin-main.log`
- 日志记录应用启动、退出、配置保存 / 恢复、窗口事件等关键信息，便于问题排查和反馈时提供上下文

**渲染错误保护：**
- 控制面板和桌宠窗口均增加渲染错误保护机制
- 控制面板发生渲染异常时显示友好错误提示，支持重试恢复
- 桌宠窗口发生渲染异常时自动降级处理，不影响透明叠加和鼠标穿透体验

### 升级说明

- v0.1.3 不涉及 userData 路径变更
- 不涉及 FFmpeg 变更
- 不涉及旧数据迁移
- 用户可从 v0.1.2 直接升级

### 使用注意事项

- 当前仅验证 Windows 系统
- 动作首次运行时需要计算 shape-cache，可能轻微卡顿，后续运行会流畅
- 开机自启可能受 Windows 权限或安全软件影响，如不生效请以管理员身份运行一次
- 反馈问题时请提供 `%APPDATA%\FurTwin\logs\furtwin-main.log` 中问题发生时间点附近的日志片段（如可能）
