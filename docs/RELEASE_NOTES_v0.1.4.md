# FurTwin v0.1.4 Release Notes

---

## FurTwin v0.1.4 - 诊断能力增强与安装兼容性修复

### Release 标题

```
FurTwin v0.1.4 - 诊断能力增强与安装兼容性修复
```

### 版本信息

- **版本号**：v0.1.4
- **类型**：诊断体验与安装兼容性更新
- **基于 Commit**：`3493d6e`

### 新增功能

**控制面板退出入口：**
- 控制面板标题栏新增「退出 FurTwin」按钮
- 关闭控制面板仍然只会隐藏窗口，不会退出 FurTwin
- 退出操作继续复用现有完整退出流程

**诊断入口：**
- 新增「打开日志目录」，打开 `%APPDATA%\Furtwin\logs`
- 新增「打开配置目录」，打开 `%APPDATA%\Furtwin`
- 打开失败时会在控制面板中显示明确错误

**主进程日志查看：**
- 支持查看主进程最近日志（`furtwin-main.log`）
- 支持手动刷新和收起
- 最多读取日志尾部 64KB，避免加载完整的大型日志文件
- 日志预览只读，不会修改、清空或锁死日志文件

### 清理与优化

**preload 调试输出：**
- 清理视频提取流程中遗留的 preload 调试输出
- 不影响提取成功和失败事件向界面的正常传递

### 问题修复

**修复安装路径包含空格时视频提取失败：**
- 修复安装目录包含空格时，ffprobe 无法获取视频分辨率导致视频提取失败的问题
- 问题曾影响 Windows 所有用户安装的默认目录：`C:\Program Files\furtwin`
- 修复后已验证：
  - FFmpeg 版本检测正常
  - ffprobe 视频分辨率探测正常
  - 中文输入视频路径正常
  - 包含空格的程序安装路径正常
  - 透明帧完整提取正常
  - 新提取动作能够加入动作库并正常播放

### 升级说明

- v0.1.3 用户可以直接升级至 v0.1.4
- userData 路径保持为 `%APPDATA%\Furtwin`
- 本版本不进行旧数据迁移
- 不升级或替换 FFmpeg / FFprobe 二进制文件
- 现有配置、动作库和日志可以继续使用
- 安装前请先完全退出正在运行的 FurTwin
- 不建议同时运行安装版和免安装版，因为它们共用同一个 userData 目录

### 使用注意事项

- 当前仅验证 Windows 系统
- 若 Windows 显示未知发布者或安全提示，请先核对下载文件的 SHA-256
- 动作首次运行时需要计算 shape-cache，可能轻微卡顿，后续运行会流畅
- 遇到问题时，可通过控制面板诊断区域打开日志目录或查看最近日志
- 反馈问题时请提供 `%APPDATA%\Furtwin\logs\furtwin-main.log` 中问题发生时间点附近的日志片段（如可能）
